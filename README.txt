README:
To run this project, copy over the folder called generative-audio-inpainting-master into the folder section of an available Google Drive account.
This folder contains everything needed to run the project including the dataset
The folder is available at this link:
https://drive.google.com/drive/folders/1p2llz_RyBvMthDI6gqBJxibXHYw2P2uw?usp=sharing
Once the data is stored on google drive, open a google colab environment available at: https://colab.research.google.com/
then, in the Notebooks folder, open the Data Exploration.ipynb file in the google colab enviroment
press run all, and click allow when prompted for permission to connect the google drive storage to the colab environment
